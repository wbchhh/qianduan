## 开发规范
1. id名称    采用驼峰命名法  #submitButton
2. 类名名称  采用横线 .nav-box-item
3. 版心      采用类名 .container   1560 - 360 = 1200
4. 重置页面默认样式   reset.css
5. 主题颜色  #65a7f3






## 编写Pc端网页遇到的问题

1. 如何根据PSD图，通过编码还原效果
： a. 需要具备html / css 基础。（合理使用标签，对常用属性的掌握）
   b. UI设计师提供的效果图，是规范的效果图。（图文：尺寸、位置）

2. 编写一个首页（基础版）大约多久才能达到业界水平
： a. 像项目中的网页（数量：1），约2小时左右完成，通常项目TAOBAO ,JD 官网首页（一天）
   b. 具体的开发时间由项目需求
   c. 在掌握html,css的前提下，提升编码速度（效率）
   
3. 初学者不知道如何根据PSD图划分页面的区块（结构）
： a. 除了反复练习，没捷径可走 (需要时间的打磨)
   b. 在网上找素材
